# Webpack Loaders Plugins
## Objective
The goal of this assignment is to deepen your understanding of Webpack loaders and plugins. You will configure Webpack to process CSS files using style-loader and css-loader and then optimize the CSS output for production using the MiniCssExtractPlugin.

## Project Overview
This project demonstrates the configuration of Webpack with loaders and plugins to process and optimize CSS files.

The project sets up Webpack with the following features:
- Processing CSS files using `style-loader` and `css-loader`.
- Optimizing CSS output for production using `MiniCssExtractPlugin`.

## Project Setup
1. Initializing Project:
    - Clone repository and initialize npm project with the `npm init -y` command.
2. Installing Webpack, Loaders & Plugins:
```bash
npm install --save-dev webpack webpack-cli
npm install --save-dev style-loader css-loader
npm install --save-dev mini-css-extract-plugin
```
3. Webpack Configuration:
```js
const MiniCssExtractPlugin = require("mini-css-extract-plugin");
const path = require("path");

module.exports = {
  entry: "./src/index.js",
  output: {
    filename: "bundle.js",
    path: path.resolve(__dirname, "dist"),
  },
  plugins: [new MiniCssExtractPlugin()],
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: [MiniCssExtractPlugin.loader, "css-loader"],
      },
    ],
  },
};
```
4. Implementing CSS & HTML Setup
    - After adding some CSS to `src/styles.css`, import `styles.css` to `src/index.js`file.
    - Link bundled Javascript and CSS files to `index.html`.
5. Building
    - In `package.json`, add build script : `"build": "webpack --mode=production"`.
    - Running build with the `npm run build` command.
